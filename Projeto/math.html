  • <math>: Elemento que encapsula expressões matemáticas em MathML.
  • <mrow>: Agrupa elementos em uma linha (linha matemática).
  • <mi>: Usado para variáveis (letras).
  • <mo>: Usado para operadores matemáticos (como =, -, +, etc.).
  • <mfrac>: Representa uma fração.
  • <msup>: Representa uma potência, onde o primeiro elemento é a base e o segundo é o expoente.
  • <msqrt>: Representa a raiz quadrada.
  • -b: Usando <mo>-</mo><mi>b</mi>, o operador de subtração é representado pelo <mo>-</mo>, e o "b" como variável é representado pelo <mi>b</mi>.
  • ± (mais/menos): O símbolo é inserido com <mo>&#x00B1;</mo> que usa um código hexadecimal para o símbolo "±".
  • √ (raiz quadrada): Envolvemos a expressão que estará dentro da raiz com <msqrt>.
